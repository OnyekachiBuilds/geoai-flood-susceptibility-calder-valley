# GeoAI Flood Susceptibility Mapping in Calder Valley

This project applies Random Forest machine learning, GIS, remote sensing, and Python to assess flood susceptibility across Calder Valley, West Yorkshire.

## Tools
- QGIS
- Python
- GeoPandas
- Rasterio
- NumPy
- Scikit-learn
- Google Earth Engine
